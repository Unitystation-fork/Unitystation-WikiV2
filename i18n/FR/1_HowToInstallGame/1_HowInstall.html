Windows

Dans mes documents/Games/ Creer le dossier “Unitystation”
Téléchargé le Zip “win” Win.zip et placer le dans le dossier précédemment créer.
Faire ensuite un clic droit sur le Zip, extraire ici. (7zip est vivement recommandé)
vous devriez avoir un exécutable “StationHub.exe”
pour crée un raccourcis sur le bureau, faitent un clic droit => envoyez vers => bureau.

GNU_Linux

[Ctrl] + [Alt] + [T]

wget  https://raw.githubusercontent.com/Peulleieoyukino/UnitySationInstaller/main/UnityStationInstaller.sh -O ~/UnityStationInstaller.sh ; sudo chmod 750 ~/UnityStationInstaller.sh ; sudo ~/UnityStationInstaller.sh

le raccourcis sera dans votre Menu/Game
